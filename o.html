Equipe

3 personnes

Temps

3 jours

Technologies

Unity 3D / C#

Context

~O~ est un projet de game jam réalisé par trois personnes pendant un week-end sur le thème "Only one matter".

Résumé

Le but de notre prototype est que sur chaque écran le joueur arrive à n'avoir qu'une seule particule.

Durant ce projet j'ai été en charge du game design, de l'implémentation sur Unity et d'une partie du code.

Téléchargez le projet ICI
